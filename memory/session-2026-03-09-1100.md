# Session 2026-03-09 (סשן שני)

## מה עשינו

### שיפור פריסת מסך הנגן
- הבעיה: כשהשיר מתקדם, הסרטון נגלל למעלה ונעלם
- הפתרון: `#screen-player` הפך ל-`position: fixed` — overlay על כל המסך
- `#player-top` = חלק עליון קשיח (סרטון + שליטה)
- `#lyrics` = `flex: 1; overflow-y: auto` — גולל בפנים בלבד
- הוספת `#search-status` ו-`#search-progress-wrap` למסך החיפוש (נפרד מהנגן)
- `setStatus` ו-`setProgress` מודעים ל-`currentScreen` ומעדכנים את האלמנט הנכון

### שיפור נדל"ן מסך
- כפתורי offset עברו מאופקי לאנכי, מוצמדים משמאל לסרטון (`#video-row`)
- הסרטון הוגבל ל-`max-height: 32vh` (במקום 42vh)
- טקסט offset קוצר מ-`offset: +0.0s` ל-`+0.0s`
